# WalterPMMD
1. Definición del proyecto: El proyecto consiste en crear un aplicación móvil para gestionar un Apiario y controlar los accesos de los empleados y usuarios al apiario según su rol en la empresa con sus respectiva identificación y clave. Además de control de empleados y usuarios, también gestionaremos y clasificaremos todos los apiarios mediante diversos parámetros y cuales son más productivos.

2. Especificaciones del proyecto: Se necesita una aplicación móvil que tenga lo siguiente:
   -  Gestión y control de acceso de empleados y usuarios (CRUD).
   -  Gestión y control de apiarios (CRUD).
   -  Introducción de datos y parámetros en tiempo real de cada apiario (Cámaras, sensores, movimiento, vibraciones).
   -  Clasificación y taxonomía de apiarios (Menús, tipología, producción de miel, disponibilidad, averías).
   -  Distribución y ubicación de cada apiario en el Mapa (API de Google Maps).

3. Tecnologías disponibles.
   - Lenguajes de programación: Java, Kotlin, XML, API Google.
   - Librerías de Android: Camera, Sensor.
   - Entornos de desarrollo: Android Studio IDE.
   - Despliegue y máquina virtuales: Dispositivo Android.
   - Diseño de interfaces: Material Design y Layouts.
   - Control de versiones y documentación: Git y Github.
